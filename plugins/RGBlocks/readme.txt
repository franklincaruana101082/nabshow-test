== Description ==

A beautiful collection of Ready-to-use Gutenberg blocks containing multiple elements which makes it easy for you to create better and awesome content.

This plugin also gives you a vast variety of Ready made Gutenberg blocks that will add more design elements for your website. It contains very useful blocks which are easy to use.

Awesome! These ready made blocks are non-registered Gutenberg Blocks.

**This plugin contains following Ready-to-use Gutenberg Blocks:**

**RGBlocks Select Block**
Provides 100+ Ready-to-use non-registered blocks and templates in popup

**MultiPurpose Gutenberg Block**
Use one block containing multiple elements

**RGBlocks Button**
Insert Button with style settings

**RGBlocks Columns**
Display columns

**RGBlocks Icons**
Display font awesome icons

**RGBlocks Countdown**
Display the count down

**RGBlocks Separator**
Add separator

**RGBlocks Accordion**
Accordion

**RGBlocks Layout**
Display layouts

**RGBlocks Heading**
Add Heading / Title

**RGBlocks Image**
Single Image

**RGBlocks Media & Text**
Add media along with text

**RGBlocks List**
Display Bullet List

**RGBlocks Slider**
Display Images Slider

**RGBlocks Social**
Insert social Buttons

**RGBlocks Tab**
Add Tabbing

**RGBlocks Gallery**
Add Gallery Images

**RGBlocks Counter**
Simple Numeric Counter

**RGBlocks Popup**
Add Popup generate

 **RGBLOCKS PLUGIN**
This plugin is recommended to use with it's origin which is a WordPress Theme - RGBlocks. It is a customizable Multipurpose Gutenberg compatible theme that lets you to easily create your website the way you want.

It's a RGBlocks Theme because it creates your website rgblocks by rgblocks.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rgblocks` directory, or install the plugin through the WordPress plugins screen directly.
