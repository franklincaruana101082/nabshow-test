== Installation ==

1. Install CometChat either via the WordPress.org plugin repository or by uploading the files to your server.
2. Activate CometChat.
3. Navigate to the CometChat tab at the bottom of your admin menu and enter your CometChat License Key to connect your CometChat account to your WordPress site.
4. CometChat will start showing immediately.
5. To customize CometChat, click on the Admin Panel button.
